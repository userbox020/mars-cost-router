# Security Policy

## Supported version

Security fixes are applied to the latest published release.

## Reporting a vulnerability

Use GitHub's private security advisory flow for
<https://github.com/userbox020/mars-cost-router/security/advisories/new>. Do not
open a public issue for an undisclosed vulnerability and do not include secrets,
credentials, private prompts, or raw session data in a report.

This repository contains static plugin metadata, policy, and instructions. It
does not package hooks, MCP servers, executable runtime code, or telemetry.
Reports about model-provider accounts or the Codex CLI itself should be sent to
the responsible upstream project or provider.
